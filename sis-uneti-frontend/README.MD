# SIS-UNETI Frontend

Este es el frontend del ecosistema SIS-UNETI, desarrollado con Angular.

## Requisitos

- Node.js 20.x
- npm 10.x

## Instalación

```bash
npm install
```

## Ejecución

```bash
npm run start
```

## Construcción

```bash
npm run build
```

## Despliegue

```bash
npm run build -- --configuration production
```
